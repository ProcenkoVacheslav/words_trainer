перед работой необходимо выполнить инструкции (Инструкции работают только на Windows):

1) Ввести в терминал команду
python -m venv .venv
2) В только что созданной папке venv открыть папку Scripts
3) Найти файл activate.ps1
4) Нажать ПКМ по файлу activate.ps1
5) В появившемся менб выбрать пункт Copy Path/Reference и выбрать пункт Absolute Path
6) Открыть терминал, нажать ctrl + v и после этого enter

если всё сделано правильно, то в консоли вместо PS "путь к исполняемому файлу" будет (venv) PS "путь к исполняемому файлу"

7) Ввести в терминал команду (требует подключение к интернету)
pip install -r requirements.txt

Если все инструкции были соблюдены, то отройте файл main.py и запустите его. Если всё заработало, то поздравляю, всё получилось.
Если нет, зайдите в файл trainer.py Если библиотека pyttsx3 подчёркнута красной линией, то введите в терминал следующую косанду
pip install pyttsx3
Это должно всё исправить.

Как пользоваться
1) Войдите в файл main.py и запустите его. Изначально будут диктоваться все словарные слова в русском языке.
2) Если хотите изменить набор слов, то перейдите в файл settings.py и замените значение FILE_NAME на путь к другому
файлу "words/имя_файла.txt"
3) Если хотите добавить свойи слова (не обязательно русские), создайте собственный файл с расширением .txt в папке words
и укажите путь на него в settings.py.
4) После того как программа озвучит слово, дождитесь появления в терминале приветственной фразы и только потом вводите слово.
Если ввести зарание, то слово не засчитается.
5) Если вас не устраивает скорость воспроизведения, измените её в settings.py
